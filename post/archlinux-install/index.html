安装

本文以<Tab>表示Tab键。

验证启动模式

如已知启动模式，该命令可忽略。

1

ls /sys/fi<Tab><Tab>

若命令显示存在efi，则系统以UEFI模式启动。若不存在，则系统可以BIOS（或CSM）模式启动。

以太网

插入网线。

Wifi

执行 iwctl 进入交互模式
然后显示交互式提示，前缀为[iwd]#
通过 station wl<Tab> connect <Tab> 选择SSID连接。

静态IP分配

检查是否已通过DHCP分配ip地址。

1

ip addr show

若无，手动分配ip地址。

1

ip addr add dev <Tab>

选择设备，<Tab> 输入IP地址，回车。

注意：IP地址应与网关处于同一子网且不得与现有设备的IP地址冲突。

ping某个测试网络连通专用网站。

1

ping -c 3 baidu.com

如果正常，继续。
更新系统时钟。

1

timedatectl set-ntp true
